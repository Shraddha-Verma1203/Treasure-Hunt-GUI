# 🏴‍☠️ Treasure Hunt GUI

> An interactive Java Swing-based Treasure Hunt Simulation that uses graph concepts, weighted paths, and destination-based treasure collection to visualize route planning and treasure discovery.

---

## 📌 Project Overview

Treasure Hunt GUI is a desktop application developed using Java Swing that allows users to create a custom treasure-hunting environment through an interactive graphical interface.

Users can create nodes, connect them using weighted edges, assign treasure destinations, and run simulations to determine routes and treasure collection outcomes.

The project combines Object-Oriented Programming principles with graph-based concepts to create an engaging and visual simulation experience.

---

## ✨ Features

✅ Interactive Java Swing GUI

✅ Create and manage nodes dynamically

✅ Add weighted directed edges

✅ Assign treasure destinations

✅ Visual graph representation

✅ Treasure hunt route simulation

✅ Simulation result visualization

✅ User-friendly interface

✅ Graph-based path management
