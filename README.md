# Treasure Hunt GUI

A Java Swing-based Treasure Hunt simulation project where users can create nodes, connect paths, assign treasures, and simulate a treasure collection route.

## Features
- Interactive GUI using Java Swing
- Add and manage nodes on canvas
- Add weighted directed edges
- Assign treasure destinations
- Simulate treasure hunt path
- Displays output and route details

## Tech Stack
- Java
- Java Swing
- Object-Oriented Programming
- Graph Concepts

## How to Run
1. Clone this repository
2. Open the project in IntelliJ IDEA
3. Open `src/TreasureHuntGUI.java`
4. Run the main class
