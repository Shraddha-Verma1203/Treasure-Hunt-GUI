## Problem Statement

Traditional graph learning can feel abstract and difficult to visualize.

This project was developed to provide an interactive environment where users can create graph structures, assign destinations, and simulate treasure-hunting routes through a visual interface.

## рҹ“Ң Project Overview

Treasure Hunt GUI is a desktop application developed using Java Swing that allows users to create a custom treasure-hunting environment through an interactive graphical interface.

Users can create nodes, connect them using weighted edges, assign treasure destinations, and run simulations to determine routes and treasure collection outcomes.

The project combines Object-Oriented Programming principles with graph-based concepts to create an engaging and visual simulation experience.
